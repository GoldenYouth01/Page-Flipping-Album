</div>

## 1. 翻页相册书（`album.html`）

一个具有 3D 翻页效果的电子相册页面。

- 逼真的 3D 书页透视与翻页动画
- 多种翻页方式：点击页角、拖拽页角、键盘左右方向键、屏幕左右滑动
- 内置背景音乐播放器，支持播放列表切换、进度跳转、上一首/下一首
- 桌面端展示为对开双页，移动端自动切换为单页模式
- 相册标题、副标题、页数、图片、音乐列表等所有内容都集中在脚本顶部的 `albumData` 对象中，直接编辑即可定制自己的相册

## 2. 音律联觉共享地图（`ambience_synesthesia.html`）

一个基于地图的协作标记工具，用户可以登录后在自己喜欢的位置添加带颜色和类别的标记，用于记录“这个地点给我什么样的感觉”。

- 注册与登录，支持创建或加入一个队伍（通过 6 位邀请码）
- 地图上点击任意位置即可新建标记，填写名称、类别、颜色与描述
- 标记分为：默认、美食、景点、住宿、交通、其他
- 地图底图可在普通地图、卫星地图、公交地图之间切换，并可叠加实时路况图层
- 每个用户只能看到与管理自己创建的标记，删除受所有权校验保护

## 入口

打开 `index.html` 即可在两个工具之间切换。
